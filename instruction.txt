1️⃣ Frontend → Node
In your Dashboard.tsx:

const API_BASE_URL = "https://pixel-1-lwbp.onrender.com";

Nothing else.
Frontend must ONLY talk to Node.

2️⃣ Node → Python
Inside your Node backend (where you call Python):
Replace localhost with:
const PYTHON_API = "https://pixel-tlxy.onrender.com";

const response = await fetch(`${PYTHON_API}/scan`, {
  method: "POST",
  body: formData
});


3️⃣ Enable CORS (or it will break)
In Node:
const cors = require("cors");
app.use(cors());

In Python (FastAPI):
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_methods=["*"],
    allow_headers=["*"],
)
